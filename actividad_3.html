Actividad 3

Actividad 3.

Uso del sensor ultrasónico.

Objetivo: Simular el control de movimiento del robot araña detectando obstáculos.
Actividad:

  • Conecta un sensor ultrasónico al Arduino para medir distancias.
  • Haz que un LED se encienda si detecta un objeto a menos de 10 cm.

Instrucciones:

  • Conecta el sensor HC-SR04:VCC al pin 5V del Arduino.
  • GND al pin GND.
  • Trig al pin digital 9.
  • Echo al pin digital 10.

Codigo para el funcionamiento del sensor ultrasónico:

// C++ code
//
#define trigPin 9
#define echoPin 10
#define ledPin 3


void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
}


void loop() {
  long duration, distance;
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);


  duration = pulseIn(echoPin, HIGH);
  distance = duration * 0.034 / 2;


  if (distance < 10) {
    digitalWrite(ledPin, HIGH); // Enciende el LED
  } else {
    digitalWrite(ledPin, LOW); // Apaga el LED
  }


  Serial.print("Distance: ");
  Serial.println(distance);
  delay(500);
}
