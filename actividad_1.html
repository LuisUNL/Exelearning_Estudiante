Actividad 1

Actividad 1.

Conociendo el Arduino Uno R3

Objetivo: Familiarizarse con los componentes principales del Arduino Uno y cómo conectarlo en un circuito.

Actividad:

  • Abrir un nuevo proyecto en Tinkercad y agregar un Arduino Uno R3.
  • Identificar sus pines: digitales, analógicos, alimentación, GND.
  • Encender un LED conectándolo al pin digital 13 sin usar una resistencia.

Instrucciones:

  • Conecta el LED al pin 13 y GND (el pin 13 ya tiene una resistencia interna).
  • Usa el siguiente código para encender el LED:

Código para la funcionalidad:

void setup() {
  pinMode(13, OUTPUT);
}


void loop() {
  digitalWrite(13, HIGH); // Enciende el LED
  delay(1000);           // Espera 1 segundo
  digitalWrite(13, LOW);  // Apaga el LED
  delay(1000);           // Espera 1 segundo
}
